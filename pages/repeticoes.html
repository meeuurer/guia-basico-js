> Repetições

As repetições (também conhecidas como loops ou laços de repetição) são estruturas que permitem executar o mesmo bloco de código várias vezes seguidas, sem que você precise escrevê-lo repetidamente. Existem diferentes tipos de loops no JavaScript, cada um ideal para uma situação:

Loop for

É o mais comum e utilizado quando você sabe exatamente quantas vezes quer repetir a ação. Ele usa um contador para controlar o ciclo.


Loop while (enquanto)

Ideal para quando você não sabe o número exato de repetições, mas sabe que deve continuar enquanto uma condição for verdadeira.


Loop forEach

Muito utilizado para percorrer Arrays (listas). Ele executa uma função para cada item da lista, sendo muito mais legível para lidar com conjuntos de dados.

Exemplo

Loop for


// Contando de 1 a 5
for (let i = 1; i <= 5; i++) {
 alert("Repetição nº: " + i);
}

• Como funciona: i = 1 (começo), i <= 5 (enquanto for verdade, continue), i++ (some 1 ao final de cada volta).



Loop while


let voltas = 0;
let energia = 100;
while (energia > 0) {
  voltas++;
  energia -= 25; // Perde 25 de energia por volta
  alert("Completei a volta " + voltas + ". Energia restante: " + energia);
}



Loop forEach


const frutas = ["Maçã", "Banana", "Morango", "Uva"];
frutas.forEach(function(fruta, indice) {
  alert(indice + ": Eu gosto de " + fruta);
})
