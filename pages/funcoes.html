Guia Básico JavaScript

> Funções

As funções são blocos de código projetados para executar uma tarefa específica. Você define uma série de instruções uma única vez e pode "chamá-las" (executá-las) sempre que precisar, em qualquer parte do seu programa.

Anatomia da Função

Para criar uma função básica, você precisa de quatro elementos:

  1. A palavra chave function.
  2. Um nome para a função
  3. Parâmetros (opcionais, ficam entre parênteses e funcionam como entradas).
  4. O corpo (o código que fica entre chaves {}).

Parâmetros e Argumentos

  • Parâmetros: São os nomes listados na definição da função. Eles funcionam como variáveis locais.
  • Argumentos: São os valores reais que você envia para a função quando a chama.

Comando return

Ele define qual será o "produto final" daquela função. Quando o JavaScript chega no return, a função para de ser executada e devolve o valor para quem a chamou.

Exemplo

function soma(n1=0, n2=0){
return n1 + n2;
}
alert(soma(2,5));
