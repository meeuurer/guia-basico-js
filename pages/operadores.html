Sobre

Operadores são símbolos especiais usados para executar operações sobre valores e variáveis.
Podemos dividi-los em algumas categorias principais de acordo com sua função:

  1. Operadores Aritméticos

    Usados para cálculos matemáticos básicos.

    • + (Adição): Soma valores ou concatena textos.
    • - (Subtração): Subtrai valores.
    • * (Multiplicação): Multiplica valores.
    • / (Divisão): Divide valores.
    • % (Resto da divisão): Retorna o que sobra de uma divisão.
    • ** (Exponenciação): Eleva um número à potência de outro.

  2. Operadores de Atribuição

    Usados para definir ou atualizar o valor de uma variável.

    • = : Atribui um valor.
    • += : Soma e atribui.
    • -= : Subtrai e atribui.

  3. Operadores Relacionais

    Usados para testar se uma condição é verdadeira ou falsa. O resultado é sempre um Booleano (true ou false).

    • == : Igual a (compara apenas o valor).
    • === : Estritamente igual (compara o valor e o tipo de dado).
    • != : Diferente de.
    • > e < : Maior que e menor que.
    • >=, <= e == : Maior, menor ou igual a.

  4. Operadores Lógicos

    Usados para combinar várias condições e criar lógicas complexas.

    • && (E / AND): Retorna true apenas se todas as condições forem verdadeiras.
    • || (OU / OR): Retorna true se pelo menos uma das condições for verdadeira.
    • ! (NÃO / NOT): Inverte o estado lógico (o que é true vira false).
