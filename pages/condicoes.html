> Condições

Sobre

No JavaScript, condições (ou estruturas condicionais) são blocos de código que permitem ao programa tomar decisões. Elas funcionam como uma encruzilhada: o código verifica se uma afirmação é verdadeira ou falsa e, dependendo do resultado, segue por um caminho diferente.

  1. Bloco if (Se)

    É a estrutura mais básica. O código dentro do if só é executado se a condição entre parênteses for verdadeira (true).

  2. Bloco else (Senão)

    O else serve para definir o que deve acontecer caso a condição do if seja falsa (false).

  3. Bloco else if (Senão se)

    Quando você tem mais de duas possibilidades, usa o else if para testar múltiplas condições em sequência.

  4. Operador Ternário

    É uma forma curta de escrever um if/else simples em apenas uma linha. É muito útil para atribuições rápidas.
    Sintaxe: condição ? valor_se_verdade : valor_se_falso

  5. O switch

    Usado quando você tem um valor específico e quer compará-lo com várias opções (casos). É mais organizado que usar muitos else if seguidos.
