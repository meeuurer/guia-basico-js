Guia Básico JavaScript

> DOM

O DOM (Document Object Model), ou Modelo de Objeto do Documento, é uma interface de programação que permite que scripts (como o JavaScript) acessem e manipulem o conteúdo, a estrutura e o estilo de um documento HTML.

Árvore DOM

O DOM organiza o HTML em uma estrutura de árvore hierárquica. Cada elemento do seu código (tags, atributos, textos) torna-se um "nó" (node) nessa árvore.

  • Document: É a raiz de tudo (o objeto document).
  • Elements: São as tags como body, h1, div.
  • Attributes: São propriedades das tags, como o href de um link ou o src de uma imagem.
  • Text: O conteúdo escrito dentro das tags.

O que é possível fazer?

Com o JavaScript manipulando o DOM, você pode:

  1. Alterar conteúdo: Mudar o texto de um parágrafo.
  2. Mudar estilos: Alterar a cor de um botão quando o usuário clica nele.
  3. Adicionar ou remover elementos: Criar novos itens em uma lista de tarefas ou deletar um comentário.
  4. Reagir a eventos: Executar uma função quando o usuário move o mouse ou pressiona uma tecla.

Exemplo

Selecionar Elementos

// Pelo ID (mais comum para elementos únicos)
const titulo = document.getElementById("meu-titulo");
// Pela Classe (retorna uma lista de elementos)
const botoes = document.getElementsByClassName("btn-azul");
// Pelo Seletor de CSS (o mais versátil)
const primeiroItem = document.querySelector("#lista .item");


Criar e Remover Elementos

// Criar o elemento
const novoItem = document.createElement("li");
// Dar conteúdo a ele
novoItem.textContent = "Novo item da lista";
// Adicionar ao HTML (dentro de uma < ul >, por exemplo)
const lista = document.querySelector("ul");
lista.appendChild(novoItem);
// Para remover:
novoItem.remove();


Alterar Conteúdo e Estilo

const box = document.querySelector("#caixa-aviso");
// Alterando o texto
box.innerText = "Operação realizada com sucesso!";
// Alterando o HTML interno (cuidado com segurança/XSS)
box.innerHTML = "Sucesso! Verifique seu e-mail.";
// Alterando o CSS diretamente
box.style.backgroundColor = "green";
box.style.color = "white";
box.style.borderRadius = "8px";
