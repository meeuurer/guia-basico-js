Guia Básico JavaScript

O que é JavaScript?

JavaScript é a linguagem de programação que está por trás de tecnologias e ferramentas que movimentam a internet. Essencial para tornar páginas web interativas e dinâmicas, junto com HTML e CSS.

Breve História

Criado em 1995 na Netscape, o JavaScript evoluiu de um script de interação simples para o padrão universal da web via ECMAScript. Com o advento do AJAX e do Node.js, a linguagem deixou os navegadores para dominar também o lado do servidor. Hoje, é o ecossistema mais popular do mundo.
